# OpenClaw Skills

智能自动化技能集合，适用于 OpenClaw AI 代理。

## 技能列表

| 技能 | 说明 | 适用场景 |
|------|------|----------|
| unified-search | 全球统一搜索引擎 | 知识查询、代码搜索 |
| smart-meme | 智能表情包系统 | 社交、娱乐 |
| find-skills | 技能发现 | 寻找可用技能 |
| openclaw-debugger | 诊断工具 | 故障排查 |
| uv-priority | Python 包管理 | 开发环境优化 |

## 安装

将技能文件夹复制到你的 OpenClaw skills/ 目录即可。
